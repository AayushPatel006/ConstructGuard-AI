# Place your construction site videos here

## Naming Convention:
- site1_construction.mp4 - Downtown Plaza
- site2_construction.mp4 - Riverside Complex  
- site3_construction.mp4 - Tech Hub Center
- site4_construction.mp4 - Harbor Bridge

## Supported formats: .mp4, .avi, .mov, .mkv

If no video files are found, the system will use the webcam feed.
